new KMeans(dataset, options, callback)

Create a K-Means.

Parameters:
Name Type Description
dataset
options options

An object describing a model's parameters:

  • k: number of clusters
  • maxIter: Max number of iterations to try before forcing convergence.
  • threshold: Threshold for updated centriod distance before declaring convergence.
callback function

Optional. A callback to be called once the model has loaded. If no callback is provided, it will return a promise that will be resolved once the model has loaded.
